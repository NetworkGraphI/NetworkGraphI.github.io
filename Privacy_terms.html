Privacy Policy

Our Privacy Policy was last updated on 8th April 2023.
This Privacy Policy describes Our policies and procedures on the collection, use and disclosure of Public data used in the project.

We use data only for the perpose of the research. In NetworkGraphI, we are committed to protecting the privacy of the people. We collect and use personal data solely using social network data accessed via API for the purpose of conducting our social network analysis research, and we will never share personal information with third parties without explicit consent.
The following is our terms of privacy.

  • Data Collection: The project will obtain data from social network APIs. This data may include personal information such as names, usernames, profile pictures, and social connections.
  • Data Use: The project will use the collected data to construct a graph for network analysis. The data will be stored on a secure server and will not be shared with third parties.
  • Data Access: Access to the collected data will be restricted to project team members who require it for the purposes of constructing the network graph and performing the analysis.
  • Data Retention: The collected data will be retained only for the duration of the project. Once the project is complete, the data will be securely deleted.
  • Data Anonymization: Personal information will be anonymized wherever possible to protect the privacy of the individuals represented in the network graph.
  • Data Security: The collected data will be stored securely using appropriate security measures such as encryption and access controls.
  • Compliance: The project will comply with all relevant privacy laws and regulations, including the General Data Protection Regulation (GDPR) and the California Consumer Privacy Act (CCPA).
